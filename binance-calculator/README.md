# binance-calculator
Simple script to calculate PnL (profits and losses) from your trades in a period

## How to Run

1. git clone
2. copy .env.example as .env
3. fill .env variables
4. npm install
5. look at the getAllOrders function inside index.js to use mock data or not
6. npm start
