# binance-trailing-stop
Trailing Stop crypto bot example, for Binance Spot.

## How to Run

1. git clone
2. cd binance-trailing-stop
3. copy .env.example as .env
4. fill .env variables
5. adjust your symbol and strategy at index.js
6. npm install
7. npm start
